# Wordcloud-Generator
_一个通过读取输入的网址的内容 或自定义的文本 生成词云的小工具_  
_A small tool that generates a word cloud by reading the content of a given URL or custom text._

</div>

## 使用方式
1.安装依赖库
```shell
pip install -i requirement.txt
```
2.开袋即食
## 特色
- [x] 自定义字体
- [x] 自定义图形
- [x] 自定义文字
- [x] 多语言支持
- [x] 网站抓取
- [ ] msn和百度系的网页暂时无法抓取，请使用自定义文本模式
